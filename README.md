# Scrape-data-from-the-web

#### 1. Implement Selectors

Use BeautifulSoup or lxml to extract product name, price, rating, and number of reviews using:

Element selectors

ID/Class selectors

Attribute selectors

(Optional) Pseudo selectors

#### 2. Extract Data from Multiple Pages

Use a loop to scrape data from at least 3 pages.

Store data in a structured format (e.g., dictionary or list of dicts).

#### 3. Extract and Clean Data Using Regex

Use regular expressions to extract structured patterns such as prices or ratings from raw text.

#### 4. Visualize the Data

Create at least two charts using matplotlib:

A bar chart showing average price per rating

A histogram showing the distribution of product prices
